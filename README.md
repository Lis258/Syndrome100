### Счетчики JaCoCO

- INSTRUCTIONS - предоставляет информацию о количестве кода, который был выполнен или пропущен;


- LINE - для всех файлов классов, которые были скомпилированы с информацией об отладке, можно рассчитать информацию о покрытии для отдельных строк. Исходная строка считается выполненной, когда была выполнена хотя бы одна команда, назначенная этой строке;


- BRANCH - вычисляет охват веток для всех операторов if и switch. Эта метрика подсчитывает общее количество таких ветвей в методе и определяет количество выполненных или пропущенных ветвей.
